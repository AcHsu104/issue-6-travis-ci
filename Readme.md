## Overview

This project demonstates the usage of [Travis CI](http://www.travis-ci.org/). It includes the following:
* Example XCode project
* Xctool
* CocoaPods
* Kiwi Testing Framework
* Travis CI configuration
* Build script (TODO)
* Coveralls integration (TODO)
